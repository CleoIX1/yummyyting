# images/

Drop your photos here. The site expects these filenames:

| File                  | Used in                          |
|-----------------------|----------------------------------|
| hero.jpg              | Hero section (right side)        |
| banana-bread.jpg      | Product card + gallery           |
| banana-bread-2.jpg    | Gallery (packaging)              |
| banana-bread-3.jpg    | Gallery (batch bake)             |
| cookies.jpg           | Product card + gallery           |
| about.jpg             | About / My Story section         |

Tips:
- Any format works: .jpg, .jpeg, .png, .webp
- Dark, moody shots look best — similar to your Instagram aesthetic
- No minimum size, but 1200px wide or taller looks sharpest
